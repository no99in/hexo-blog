为什么选择 python 来作为设计模式的示例代码

首先是语言入门难度

  1. C 语言缺乏一些面向对象的特性
  2. CPP 和 Java 入门门槛比较高

其次 Python 语法简单同时面向对象，但是由于语法的一些特性，在继承和接口方面可能需要大家去琢磨，这些需要打磨的地方将会在后面涉及到的时候讲清楚。继承和接口并没有一些强语言像 Java 的类间关系清晰。

python 环境和集成开发环境

选择 python 需要配置环境，我这里是 windows 10 pro，安装 python 相对其他系统较复杂。

下载 python 网址 https://www.python.org/downloads/release/python-372/

我使用的是 3.72 exe 版本，但是理论上，我的代码可以在 3.x 的版本上跑起来。新手不推荐 zip 的版本，这个版本甚至没有 pip 可能会让你措手不及。

安装默认即可。

其次，我选择了 pycharm 作为集成开发环境。可能大家会说有点大材小用，但是这里我想强调，我一般不会让大家安装一个臃肿的开发环境，下面我会给你几张截图，来告诉你为什么使用它。
