安装 git (install git)

git 是啥？

(what is git)

git是一个版本控制系统，说白了其实将你的代码每次保存后记录下来，已恢复到上一个版本。

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

git 与 hub？

(git and github)

github是一个开源社区，大家使用git将自己的代码上传到github与大家分享。

GitHub is a software source code hosting service that USES Git for version control

下载

(downlaod)

  1. 官网 https://git-scm.com/
    (The website https://git-scm.com/)

  2. 根据你的视窗位数选择一个合适的git
    (Choose a git that is appropriate for your windows bits)

安装

(install)

安装比nodejs复杂一些，这里将他记录下来分享给大家。

Installing git is a bit more complicated, so I’ll share the process with you.

  1. 取消 git bash here 这会让你的右键菜单中莫名其妙多出来两项无关项。
    (Undo git bash here will cause two new items to appear in your right-click menu for this reason.)

  1. 选择使用windows的console。
    (Choose to use the Windows console)

其他按照默认即可。

Others can be as the default.
