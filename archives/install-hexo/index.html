安装 hexo (install hexo)

hexo 是啥

(What is hexo?)

hexo是一个轻量级的博客，他最大的优点就是可以轻松部署到Github、码云等平台。

Hexo is a lightweight blog system, and the great thing about it is that hexo is easy to deploy to Github, Gitee and some such etc.

为什么选择个人博客

(Why choice hexo)

博客平台辣么多，为什么选择个人博客。

Blog platform is so much, why choose personal blog.

  1. 自主可定制 self-customization

  2. 绑定自己的域名 Bind your own domain name

  3. 没有太多广告干扰 No AD

  4. 专注于记录与阅读 Focus on recording and reading

使用说明

(usage)

怎么安装使用hexo是一个问题。

How to install hexo?

git + nodejs

可以参阅我以前的文章安装这些软体。

See my previous articles on installing these software.

安装hexo脚手架

(install hexo-cli)

npm install hexo-cli -g

选择安装博客的目录

(Select the directory to install the blog)

cd ~/Documents/hexo-blog

我选择的是个人文档目录下的hexo-blog，这个根据你的喜好而定。

I chose the personal document(~/Documents/hexo-blog) as directory,In fact this is up to you.

初始化博客并且进行安装相关依赖

(Initialize the blog and install the dependencies)

hexo init
npm i

测试运行你的博客

(run and enjot your blog)

hexo g
hexo s

在本地打开localhost:4000进行对内容的调试，下一节我将会告诉大家如何将书写markdown博客和将博客部署到代码托管平台，大家都可以根据你的网址对你的网站进行访问，而这一切，均是免费！

Open localhost:4000 locally to debug your blog. In the next section, I will tell you how to write markdown archives and deploy the blog to the code hosting platform. Everyone can visit your website according to your website address.
