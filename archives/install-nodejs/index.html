安装 nodejs (install nodejs)

nodejs 是啥玩意？

(What is nodejs?)

Node.js® 是一个基于 Chrome V8 引擎 的 JavaScript 运行时。

Node.js® is a JavaScript runtime built on Chrome’s V8 JavaScript engine.

看完这个，我是一脸蒙蔽。

After watching the introductio, I was froze.

nodejs其实就是运行在服务器上的js。

In fact, Nodejs is javascript running on the server.

下载

(install)

  1. 到官网https://nodejs.org/zh-cn/。
    (Go to the official website of nodejs at https://nodejs.org/.)

  2. 下载长期支持版。
    (Download the long-term support version.)

安装

(install)

安装过程比较简单，基本上一路默认即可。

Installation process is relatively easy, just keep default.

测试

(check-out)

终端输入node -v即可。
input node - v on Terminal.
